# Auto-InstallTools
#### here you can download 4 tools for termux.

---
## Installing for termux

* `pkg install git`
* `git clone https://github.com/mishakorzik/Auto-InsTools`
* `cd Auto-InsTools`

## Installing for linux

* `pkg install git`
* `git clone https://github.com/mishakorzik/Auto-InsTools`
* `cd Auto-InsTools`

#### Enter a command to start the AutoInstalling.
